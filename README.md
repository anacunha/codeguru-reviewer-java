# java-project-aws

Use the following command to create a new directory called `myapp` with a project configuration file :

```shell
mvn -B archetype:generate \
 -DarchetypeGroupId=org.apache.maven.archetypes \
 -DgroupId=com.example.myapp \
 -DartifactId=myapp
 ```
